Desarrollo Web 1

Descripción del Curso

Este curso constituye la primera etapa en la formación de desarrollo web, abarcando los fundamentos esenciales de HTML, CSS y JavaScript, que son los pilares de toda aplicación web moderna.

El curso permite comprender la arquitectura de aplicaciones front-end, el papel de cada tecnología en la creación de aplicaciones modulares, y conocer las mejores prácticas de diseño, así como identificar prácticas desventajosas en el desarrollo de software.

Contenidos del Curso

  • Introducción al HTML: Elementos, semántica y atributos
  • Hojas de Estilo en Cascada (CSS): Selectores, propiedades y modelo de caja
  • Maquetación de Sitios Web: Posicionamiento, diseño responsivo, Flexbox y CSS Grid
  • Frameworks de Diseño: Bootstrap, mallas y componentes
  • Introducción a JavaScript: Sintaxis, estructuras de control, funciones y POO
  • DOM y JavaScript Asíncrono: Manipulación del DOM, JSON, AJAX y consumo de APIs

Guías de Ejercicios

Guía 1: Fundamentos HTML y CSS

Identificación de la estructura básica de documentos web. Aplicación de reglas de estilo CSS mediante estilos embebidos y hojas de estilo externas. Utilización de selectores CSS básicos y propiedades para la manipulación de fuentes y textos.

Ejemplo 2 Estilos Embebidos Interstellar Países

Guía 2: Selectores Avanzados y Modelo de Cajas

Utilización y combinación de selectores CSS avanzados. Conocimiento de la utilidad de pseudo-clases y pseudo-elementos CSS. Creación de páginas web utilizando el Modelo de Cajas, identificando sus partes: margin, border, padding y content.

El Discurso del Rey Tablero de Damas Mi Biografía

Guía 3: Posicionamiento y Maquetación Web

Utilización de diferentes tipos de posicionamiento CSS y diferenciación apropiada entre ellos. Aplicación de propiedades CSS relacionadas con la visualización de elementos. Maquetación de sitios web con diseños fijos y fluidos.

Oscares Menú Horizontal Viajeros Cartas

Guía 4: Diseño Web Responsivo

Creación y estilización de formularios utilizando controles de HTML5. Uso correcto de la meta-etiqueta viewport. Definición de media queries para aplicar reglas de estilo según el ancho del viewport. Construcción de páginas web responsivas aplicando los conocimientos adquiridos durante el curso.
